Time Series Forecasting with LSTMs for Daily Coronavirus Cases using PyTorch in Python

  • Tech Stack: PyTorch, Deep Learning
  • Machine Learning Library: PyTorch
  • Concepts Used: Deep Learning, Time Series Forecasting, LSTMs
  • Programming Language: Python
  • Data Visualization: Matplotlib, Seaborn
  • Data Analysis: Pandas, NumPy
  • Version Control: Git
  • Github URL: Project Link

This project demonstrates the use of PyTorch for Time Series Forecasting with Long Short-Term Memory (LSTM) networks to predict daily coronavirus cases. The dataset is sourced from the Johns Hopkins University Center for Systems Science and Engineering (JHU CSSE) and includes the number of reported daily cases by country. The LSTM model is trained on the confirmed cases to predict future trends.

Daily Cases Dataset

The dataset is provided by JHU CSSE and is regularly updated. It includes the number of reported daily cases by country. The data is preprocessed to focus on confirmed cases, and the cumulative nature of the data is adjusted to represent daily cases.
